Raspberry Pi GPIO connections

GPIO 12: PWM Heater
GPIO 13: PWM Fan
GPIO 4: 1-wire temperature sensors

Temperature Sensor Mapping

In order to read a temperature from the temperature sensors multiple IDs can be found in : /sys/bus/w1/devices/. So to read a value from a sensor, one can do: cat /sys/bus/w1/devices/10-0008039ad4ee/w1_slave.

Each ID in the devices folder correspond to a particular sensor. Follow the wires of the temperature sensors and you will find a paper label with a number on it. The ID to number mapping is:
